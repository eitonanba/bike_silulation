# Bike Map Simulator - Mobile Free Drive

航空写真上をバイクで走る簡易ブラウザシミュレータです。

## 起動

`bike_map_simulator.html` をスマホまたはPCのブラウザで開いてください。

## スマホ操作

- 画面左半分を押す: 左へ曲がる
- 画面右半分を押す: 右へ曲がる
- 画面下の `← 左` / `右 →` ボタンでも操作できます
- `戻る`: スタート位置へ戻る
- `一時停止` / `再開`: 走行の停止・再開
- `−` / `＋`: ズーム調整
- `表示`: 詳細表示の開閉

## PC操作

- A / ←: 左へ曲がる
- D / →: 右へ曲がる
- R: スタート位置へ戻る
- Space / P: 一時停止・再開
- + / -: ズーム

## 仕様

- バイクは自動前進します
- 操作は左右だけです
- バイクの進行方向が常に画面上になるように、マップ側が移動・回転します
- 通行不可領域はありません。マップ内はどこでも走行可能です
- スタート位置と向きは `スタート指定` からドラッグで変更できます
