# Bike Map Simulator - Free Drive

- Open `bike_map_simulator.html` in a web browser.
- The bike always moves forward toward the top of the screen.
- Use `A` / `Left Arrow` to turn left.
- Use `D` / `Right Arrow` to turn right.
- Use `R` to return to the start position.
- Use `+` / `-` to zoom.
- Click `Start position` and drag on the map to set the start point and direction.
- No impassable regions are enforced; the bike can move across the whole map image.
